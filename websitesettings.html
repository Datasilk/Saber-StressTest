Prepare Stress Test

Before you can run your stress test, you must first prepare for the test. This will generate 100 web pages in your website that will be used for the stress test.

Run A Stress Test

By running a stress test, you will open a new browser tab that will load random test web pages inside 24 iframes as quickly as possible. As soon as each iframe loads, they will redirect to another random test web page immediately afterwards. Once you load 500 test web pages, the test results will appear.

Run Test
Delete Stress Test

Once you are done with your stress test, you can delete all web pages associated with the test.
